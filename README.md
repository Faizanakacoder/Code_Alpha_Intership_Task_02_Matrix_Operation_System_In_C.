## 📌 Project Overview

The **Matrix Operations System** is a console-based application developed in **C Programming Language** as part of the **CodeAlpha Virtual Internship Program**.

This project performs essential matrix operations including **Matrix Addition**, **Matrix Multiplication**, and **Matrix Transpose** using **Functions** and **2D Arrays**. The application provides a simple menu-driven interface that allows users to perform different matrix calculations efficiently.

The project helps in understanding matrix manipulation techniques and strengthens knowledge of arrays, functions, loops, and user-defined programming concepts in C.

## 🚀 How to Run

### Step 1: Clone Repository

```bash
git clone https://github.com/your-username/matrix-operations.git
```

### Step 2: Navigate to Project Folder

```bash
cd matrix-operations
```

### Step 3: Compile

```bash
gcc matrix_operations.c -o matrix
```

### Step 4: Run

#### Windows

```bash
matrix.exe
```

#### Linux / Mac

```bash
./matrix
```
